# Fullstack Materiais

Repositório de materiais didáticos para o curso **Desenvolvimento Fullstack com Node.js**.

## Ementa do Curso

| Aula | Tema |
|------|------|
| 01 | Introdução ao Node.js e ao backend |
| 02 | Instalação, configuração e primeiros scripts Node.js |
| 03 | Fundamentos do Express: estrutura, rotas e servidor básico |
| 04 | Middlewares: conceito, uso e exemplos práticos |
| 05 | Manipulação de requisições e respostas (req/res) |
| 06 | Rotas dinâmicas e parâmetros |
| 07 | Controllers e organização de código |
| 08 | CRUD básico com dados em memória |
| 09 | Introdução ao MongoDB e integração com Mongoose |
| 10 | CRUD completo com MongoDB |
| 11 | Validação de dados e tratamento de erros |
| 12 | Autenticação: JWT, login e proteção de rotas |
| 13 | Autorização e controle de acesso |
| 14 | Upload de arquivos e manipulação de imagens |
| 15 | Testes automatizados: Jest/Supertest |
| 16 | Documentação de API: Swagger/OpenAPI |
| 17 | Deploy: Heroku/Vercel/Azure |
| 18 | Boas práticas, arquitetura e padrões REST |
| 19 | Projeto final: planejamento e desenvolvimento |
| 20 | Apresentação, revisão e avaliação do projeto final |

## Estrutura do Repositório

## Como compilar os slides

```bash
cd aulas/aula01
pdflatex aula01.tex
```

Requisitos: TeX Live ou MiKTeX com pacotes Beamer instalados.
